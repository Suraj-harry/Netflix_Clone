# 🎬 Netflix Clone

This project is a Netflix-inspired web application developed using React and Vite. It allows users to sign up, log in securely, and browse movies and TV shows using data from the TMDB API. The main purpose of building this project was to improve my frontend development skills and gain practical experience with React, Firebase Authentication, and API integration.

---

## Features

- User Registration and Login
- Firebase Authentication
- Browse Trending Movies
- Browse Popular TV Shows
- Responsive User Interface
- Dynamic Movie Banners
- Real-time Data from TMDB API
- Clean and Simple Design

## Getting Started

Clone the repository

```bash
git clone https://github.com/Suraj-harry/Netflix_Clone.git
```

Go to the project folder

```bash
cd Netflix_Clone
```

Install the required packages

```bash
npm install
```

Run the project

```bash
npm run dev
```

---

## How the Project Works

After signing in, the application fetches movie and TV show data from the TMDB API and displays it in different sections. React components are used to organize the interface, while Firebase handles user authentication. The application is responsive, making it easy to use on both desktop and mobile devices.
